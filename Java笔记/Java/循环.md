## 四种循环
- do while
- while
- for
- for( :)   ，，增强循环，本质上也是使用iterator迭代器，所有集合类都默认实现iterator接口，实现此接口则能增强for循环能力，for-each
- Iterator iterator = linkedList.listIterator();    while (iterator.hasNext())   ，， Iterator迭代器循环，iterator()方法，会要求返回一个iterator的接口子类实例对象，此接口包含：hasNext(),  next()
	- iterator()方法始终通过cursor和laster的指针来获取元素值及向下的遍历索引
	- 迭代器有向前和向后两种遍历方式，当使用向前遍历时要保证指针在迭代器的结果，否则无法获得结果值
