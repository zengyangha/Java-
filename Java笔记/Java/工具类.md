# 点此回到[[JAVA（由此开始）]]

## Collections
- addAll   批量添加
- .sort（）排序
- .binarySearch（）二分查找，如果该数组或集合是无序的那前提要排序sort，该查找方式是每次查找先将搜索范围缩小一半

## Arrays
- 提供数据操作的工具类
	- 排序（Sorting）
	- 搜索（Searching）
	- 比较（Comparison）
	- 填充（Filling）
	- 复制（Copying）
	- 转换为列表（Converting to List）
	- 输出为字符串（Output to String）
	- 哈希码（Hashing）
