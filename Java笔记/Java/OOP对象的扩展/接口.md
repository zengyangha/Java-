# 点此回到[[OOP对象]]

## has-a关系      关键词：implement   接口用interface表示，如：public interface Fan(){}

## 接口具有  约定  、规定的功能，体现在接口的名称（形参）和注释上

## 一般用接口最多，继承留给最重要的父类

## 在接口中，所有权限都是    public   ，所有方法都是           abstract    抽象的 ，所有的变量都是    静态常量（static）

## 接口不能实例化

## 接口代表一种能力，一个类实现了该接口，那这类就具有该能力

## 接口只能继承接口

## 抽象类可以实现接口，而且不重写该抽象方法
